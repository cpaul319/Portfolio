Below are descriptions of some of my work. If you click the photo you will see a screenshot of the app being used.

Links to the actual apps and to the GitHub repos are available below as well.

Got Swag

Full Stack E-commerce site where fans of Game of Thrones can buy merchandise.
This site was built using React.js, MySql/Sequelize, Node.js, Express.js, several React npm packages including Reactstrap, and utilizes the PayPal API.

Hit The Trails

Was designed from the beginning to be a mobile application. It uses the cell phone location (with user's permission) to find the nearest hiking Trails. Technologies used for this project were jQuery, Shards UI kit (Bootstrap), Ajax, CSS Grid, Firebase and mobile responsive. APIs: Hiking Project, Open Weather and Google Maps

Confessions

Full Stack web application where users can post "Confessions" and other users can guess if the confession is true or not. This site was built using Handlebars.js, MySql/Sequelize, Node.js, Express.js, Express-Sessions and Bootstrap.

Crytsals Collector

A fun and interactive game for web browsers. Technologies used were CSS3, HTML5, BootStrap, JavaScript and jQuery .
Spooky Friend Finder

This full stack site compares users surveys then displays the name and picture of the user with the best overall match. Technologies used:CSS3, HTML5, BootStrap, JavaScript, Express.js and Heroku
